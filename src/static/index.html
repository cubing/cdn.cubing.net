cdn.cubing.net

This site is a CDN ("content delivery network"), designed to serve JavaScript and other web resources in way that is:

  • fast — quick to load from anywhere in the world,
  • efficient — avoid downloading too much code if you don't need it,
  • convenient — use directly on any web page without any special setup.

Visit js.cubing.net to learn how to use cdn.cubing.net for your cubing website or web app.

Available URLs

The following is a list of all available URLs, ready to be used in browsers. These URLs correspond to npm packages using node module resolution, and can be imported in JavaScript/CSS files directly. For example, the following will import the same functionality:

 From npm  import { Alg } from "cubing/alg";

 From the CDN  import { Alg } from "https://cdn.cubing.net/js/cubing/alg";

If you're not certain what this means, we recommend using the examples from js.cubing.net.
